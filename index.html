Welcome to the Fibonacci Generator!

Please input a number between 1 and 20

0 inputted is nothing, 1 inputted is [0], 2 inputted is [0, 1]

3 inputted is [0, 1, 1], 4 inputted is [0, 1, 1, 2]

5 inputted is [0, 1, 1, 2, 3], 6 inputted is [0, 1, 1, 2, 3, 5]

What's happening here is, for example 6 inputted was [0, 1, 1, 2, 3, 5, 8]

We can anticipate the next number for group 7 to be the last 2 added together

Then stuck onto the back of the array 3 + 5 = 8

Therefore 7 inputted is [0, 1, 1, 2, 3, 5, 8], and there are 7 numbers in the array.
